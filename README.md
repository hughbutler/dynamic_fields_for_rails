# DynamicFieldsForRails

Dynamic fields helper for Rails.

## Installation

Add this line to your application's Gemfile:

    gem 'dynamic_fields_for_rails'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install dynamic_fields_for_rails
